# Hostel Allocation System

A CRUD web application using **JSP + JDBC + MySQL** to manage hostel room allocation.

## Features
- Add / Edit / Delete Students
- Allocate rooms dynamically
- View student-room mapping

## Tech Stack
- JSP, JDBC, Servlet
- MySQL
- Apache Tomcat

## Setup
1. Import project into Eclipse
2. Add MySQL Connector JAR in /lib
3. Create DB using `sql/hostel_db.sql`
4. Deploy on Tomcat 10
